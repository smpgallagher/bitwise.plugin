Reports duplicate constant values in the companion object of a Kotlin class annotated with @AutoMask.

The inspection evaluates integer and bitwise constant expressions, then highlights constants whose resolved values match an earlier constant in the same companion object.

Duplicate mask values can make generated masks ambiguous or cause the same bit flag to be used for multiple meanings. 


@AutoMask
class Permissions {
    companion object {
        const val READ = 1 shl 0
        const val WRITE = 1
    }
}
